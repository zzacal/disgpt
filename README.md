# Disgpt
A Chat GPT Discord bot

## Running locally

### Install
- Install node
- Install nodemon
- Install ngrok
- Install typescript

### Run
- `npm run dev`
- `ngrok http 3000`
- take the address from ngrok and stick it in Discord > Your App > General Information > INTERACTIONS ENDPOINT URL `https://123-32-444-fs-2ds.ngrok.io/interactions`
